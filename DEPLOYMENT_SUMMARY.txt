KEY ENVIRONMENT VARIABLES
───────────────────────────────────────────────────────────────────────────────

Supabase (from Supabase Dashboard → Settings → API):
  NEXT_PUBLIC_SUPABASE_URL=https://xxxx.supabase.co
  NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJ...
  SUPABASE_SERVICE_ROLE_KEY=eyJ...

Clerk (from Clerk Dashboard → API Keys):
  NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
  CLERK_SECRET_KEY=sk_test_...

Encryption (from: openssl rand -hex 32 / openssl rand -hex 16):
  ENCRYPTION_KEY=0123456789abcdef... (32 bytes / 64 hex chars)
  PSEUDONYM_MAP_ENCRYPTION_KEY=0123456789abcdef... (32 bytes / 64 hex chars)
  CRON_SECRET=0123456789abcdef (16 bytes / 32 hex chars)
